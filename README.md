# Chip-configurator
Core APB 3 Configurator

This project is a Qt-based application designed to configure and manage APB3-based systems.
It provides a user-friendly interface for working with configuration parameters and generating the required outputs.

The project was completed on 30.12.2022.

🛠 Requirements

Before building the project, make sure you have:

Qt libraries installed (Qt 5 or Qt 6 recommended)

A working C++ compiler (GCC / Clang)

make installed on your system

🚀 Build Instructions

Download and install the Qt libraries from the official Qt website.

Clone or download this repository.

Open a terminal and navigate to the project directory:

cd core_apb_3_configurator

Build the project by running:

make

After a successful build, the application will be generated and ready to run.
