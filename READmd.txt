Build Instructions

To build and run this project, please follow these steps:

Install the Qt libraries (make sure Qt is properly configured and added to your system path).

Navigate to the project directory:

core_apb_3_configurator

Open a terminal in this directory.

Run the following command:

make

The build process will start automatically, and all components of the program will be compiled.
